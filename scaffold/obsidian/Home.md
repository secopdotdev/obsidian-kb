## System health — at a glance

> Project counts by RAG status. Drill into the colored sections below.

```dataview
TABLE WITHOUT ID key AS "Health", length(rows) AS "Count"
FROM "02-projects"
WHERE type = "project"
GROUP BY file.frontmatter["rag-flag"]
```

## On Fire — needs attention now

```base
filters:
  and:
    - note.type == "project"
    - note["rag-flag"] == "red"
formulas:
  grp: note.classifier
  next: note["next-action"]
properties:
  file.name:
    displayName: Project
  formula.grp:
    displayName: Group
  formula.next:
    displayName: Next action
views:
  - type: table
    name: OnFire
    order:
      - file.name
      - formula.grp
      - formula.next
```
